# Design Specifications — 学术 PPT 硬性规范

本文件是**所有模板共享的底层规范**，任何 template / fragment / example 都不得违反。

---

## 1. 画布

| 参数 | 值 |
|---|---|
| 画幅 | 16:9 |
| 分辨率 | 3840×2160（4K） |
| 背景 | 纯白 `#FFFFFF`，不得有任何渐变、纹理、底图、水印 |
| 外边距 | 60 px（四周） |
| 栏间距 | 15 px gutter |
| 栅格 | 24 列 |

---

## 2. 配色（严格白名单）

| 角色 | HEX | 用途 |
|---|---|---|
| Primary Red | `#C0392B` | 主红，用于 header bar、强调关键词、引出线 |
| Light Red Fill | `#FDEDEC` | 公式框、高亮条的浅红填充 |
| Dark Gray Text | `#2C3E50` | 正文文字、主标题 |
| Mid Gray | `#7F8C8D` | 辅助文字、副标注 |
| Border Gray | `#BDC3C7` | 卡片边框、分割线 |
| Ghost Gray Fill | `#ECF0F1` | 非编号 chip 的底色 |

**可用的次级装饰色**（仅限点云 / 图表 / 少量数据可视化，绝不用于大面积底色）：

| HEX | 用途 |
|---|---|
| `#2980B9` | 点云深蓝 |
| `#16A085` | 点云青 |
| `#F1C40F` | 点云暖黄 |
| `#27AE60` | 状态"正常"绿 |
| `#F39C12` | 状态"预警"橙 |

禁止：紫色、粉色、渐变、霓虹、卡通高饱和色。

---

## 3. 字体与字号

字体一律使用 **思源黑体（Source Han Sans SC）**。

| 层级 | 字号 | 字重 | 颜色 |
|---|---|---|---|
| 顶部横幅主标题（Template A） | 40pt | Bold | white on #C0392B |
| 左上页面主标题 | 44pt | Bold | #2C3E50 |
| 子标题 / 小节号 | 32pt | Bold | #2C3E50 或 #C0392B |
| 模块 header（红条白字） | 22–24pt | Bold | white on #C0392B |
| 正文摘要段 | 20pt | Regular | #2C3E50 |
| 模块内小节标题 | 16–18pt | Bold | #2C3E50 |
| 项目符号正文 | 14–16pt | Regular | #2C3E50 |
| 引出线标注 | 13–14pt | Regular | #2C3E50（编号 `(n)` 用 #C0392B） |
| 规格 / 数据标注 | 12pt | Light | #7F8C8D |
| 底部"关键科学问题"句 | 26pt | Bold | #C0392B |

数学 / 公式：标准斜体，字号与正文一致。

---

## 4. 形状与阴影

- 卡片圆角：**8 px**（Template A 圆角芯片）或 **0–2 px**（Template B 平底矩形）。
- 引出线标注框：4 px 圆角，1 px 红边 `#C0392B`，白底。
- 卡片边框：1 px `#BDC3C7`。
- 阴影：仅 Template A 使用极轻阴影 `y=2, blur=6, opacity 8%`；Template B / C 默认 **无阴影**，保持扁平学术风。
- 红色横幅 / header bar 内部无描边。

---

## 5. 引出线 / 箭头

- 引出线颜色：`#C0392B`
- 粗细：**1 px hairline**（机器人部件标注）或 **2–3 px**（流程箭头）
- 端点：**4 px 实心红圆点**锚定在部件上
- 头部：**实心三角箭头**（流程箭头）或 **直线无头**（标注）
- **引出线禁止交叉**，必须做辐射式或 L 型布线
- 流程箭头可附文字标签（贴在箭身中段，白底红边小框）

---

## 6. 图标

- 统一使用**线性图标（line icon）**，2 px 描边
- 颜色：红 `#C0392B` 或 深灰 `#2C3E50`
- 主要图标 40×40 px；辅助图标 24–28 px
- **禁止** 使用彩色填充 / 写实 / 3D 图标

---

## 7. 语言规则

- 所有屏幕可见文字：**简体中文**
- 硬件 / 模型名允许英文原名（如 ZED 2, Livox Mid-360, NVIDIA Jetson Orin）
- 公式 / 符号：英文 + 斜体数学
- 提示词本体：**英文为主**，中文字面值保留中文原样

---

## 8. 布局层级约束

每张幻灯片从上到下**必定**包含以下层级（具体实现因模板而异）：

1. **标题层**（顶部横幅 或 左上大小标题）
2. **摘要层**（1 段介绍性文字，含高亮关键词）
3. **主体层**（2–4 个模块，等宽或自适应栅格）
4. **收束层**（关键科学问题 / 核心贡献 / 底部强调条，可选）

**必须丢弃的层**：
- ❌ 文献 citation 行
- ❌ 页码
- ❌ 作者姓名
- ❌ 机构水印（除非用户显式要求且放置于右上角 32×32 灰度 logo 占位）
- ❌ 装饰性 emoji

---

## 9. 学术性语言基线

摘要段、模块小标题、关键科学问题的用词必须符合以下风格：

- ✅ "实现 / 构建 / 面向 / 机理 / 协同 / 表征 / 映射 / 同步对齐"
- ✅ 动词 + 量化 + 名词化结构
- ❌ "炫酷 / 强大 / 领先 / 颠覆 / 一站式 / 全方位" 等营销词
- ❌ 感叹号、反问句、口语化表达

---

最终交付给用户的提示词**必须**用如下结构：

````
```text
Create a highly detailed professional academic presentation-style image ...
================================
PAGE STRUCTURE
================================
...
================================
MODULE 1 / 2 / 3 / ...
================================
...
================================
GLOBAL DESIGN SPECIFICATIONS
================================
...
```
````

代码块外用中文简述本次版本的关键改动与建议的迭代点。
